Core Features

🔊

Wake Word Detection

Always-on background listener activates only when it hears Hey Sai, Hello Sai, or Ok Sai.

🌐

Multi-Language Recognition

Supports English, Hindi, Telugu, Tamil, and Kannada with automatic microphone detection and noise filtering.

🧠

AI Intent Engine

Converts natural speech into structured intents. Backed by Ollama (local LLM) or OpenAI GPT with an offline fallback.

🖥️

Windows Control

Open apps, create folders, search the web, control media playback, take screenshots, shutdown or restart.

📱

Android Companion

Sai Companion app connects over Wi-Fi REST API to send SMS, make calls, scroll feeds, and open apps on your phone.

🗓️

Reminders

Set time-based reminders with natural language. Sai speaks and pops up an alert when the time arrives.

🧩

Plugin Architecture

Drop Python files into the plugins/ folder and they load automatically. Ships with Weather, Jokes, and Time plugins.

💾

Memory System

"Sai, remember my favourite language is Python." Stored in SQLite and recalled conversationally at any time.

🔐

Voice Authentication

Enrol your voice print. Commands from unknown voices are rejected, keeping your assistant private.

Architecture

Fully modular, multi-threaded design for fast, reliable responses.

Installation

1

Clone the repository

git clone https://github.com/kskreddy2k7/KSKR-AI-Voice-Operating-System.git
cd KSKR-AI-Voice-Operating-System
2

Run the installer

python install.py

This installs all Python dependencies, sets up directories, and verifies your microphone.

3

Optional – Install Ollama for local AI

# Download from https://ollama.com
ollama pull llama3
4

Launch Sai AI

python main.py           # GUI mode
python main.py --no-gui  # headless CLI
python main.py --enrol   # enroll your voice

📱 Android Companion

  1. Set "android": {"enabled": true} in config/settings.json.
  2. Build the APK from android_app/ using Android Studio.
  3. Install the APK on your Android phone.
  4. Connect the phone to the same Wi-Fi network as your PC.
  5. Enter your PC's IP address in the Sai Companion app.
